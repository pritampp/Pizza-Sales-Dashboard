# 🍕 Pizza Sales Analysis Dashboard (Power BI)

This project presents an end-to-end Business Intelligence solution built using Power BI to analyze pizza sales performance across multiple dimensions including revenue, order trends, product categories, and customer behavior patterns.

# 🔍 Key Highlights
Developed an interactive Power BI dashboard to monitor sales KPIs such as Total Revenue, Total Orders, Average Order Value, and Sales Growth.
Designed a clean, modern UI with a consistent color theme and card-based layout for improved readability and user experience.
Implemented dynamic filtering (Pizza Type slicer) to allow users to explore data across different product segments.

# 📊 Insights Delivered
- Sales Trend Analysis: Identifies monthly revenue patterns, highlighting peak sales periods and seasonal fluctuations.
- Order Distribution by Hour: Reveals peak business hours (afternoon to evening), helping optimize staffing and operations.
- Category Performance: Shows revenue contribution across categories (Classic, Supreme, Chicken, Veggie), enabling product-level strategy decisions.
- Top-Selling Products: Highlights high-performing pizzas driving the majority of sales volume.

# 🛠️ Technical Implementation
- Built a data model integrating multiple datasets (orders, order details, pizzas, pizza types) using relationships.
- Created optimized DAX measures for key metrics such as Total Revenue, Average Order Value, and Sales Growth.
- Applied data transformation and modeling best practices to ensure performance and scalability.
- Designed visuals using bar charts, line charts, KPI cards, and slicers for effective data exploration.

# 🎯 Business Value

This dashboard helps business users to:
- Monitor overall sales performance in real-time
- Identify high-performing products and categories
- Understand customer ordering patterns
- Make data-driven decisions to improve revenue and efficiency
